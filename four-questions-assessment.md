## 问题1：人口消耗和建筑维护费 —— 玩家怎么知道？

### 📋 现状审计

| 消耗类型 | 数值显示 | 显示位置 | 可见性评级 |
|---------|---------|---------|-----------|
| 人口食物消耗 | ✅ `-X.X🟢/s` | 状态面板 `rowPopFood` 行 | ⭐⭐⭐ 已有，始终可见 |
| 功率水平 | ✅ `XX%` | 状态面板 `rowPowerLevel` 行 | ⭐⭐⭐ 已有，颜色分级 |
| 维护费总量 | ✅ `-X.X⚡ -X.X🟢` | 状态面板 `rowMaintenance` 行（P2+） | ⭐⭐⭐ 已有 |
| 维护费管理开销 | ✅ `(+X%开销)` | 同上 | ⭐⭐ 有但不够直观 |
| 单建筑维护费 | ✅ tooltip中显示 | 悬停建筑格子 | ⭐⭐ 需要主动悬停 |
| 资源竞争状态 | ✅ `⚖️ 供需` | 状态面板 `rowCompetition` 行（P3+） | ⭐⭐⭐ 已有 |
| **资源速率分项明细** | ❌ 只显示净值 | 资源卡片 `+X.X/s` | ⭐ **问题所在！** |

### 🎯 核心问题

**净值隐藏了消耗细节。** 玩家在资源卡片上只能看到 `+3.2⚡/s`（净值），但**不知道**这个数字是怎么来的：
- 产出 `+5.0⚡`，建筑消耗 `-0.8⚡`，维护费 `-0.7⚡`，人口 `-0.3⚡` → 净 `+3.2⚡`

玩家会问：**"为什么我的能量增速变慢了？是维护费？人口？还是建筑消耗？"**

### 💡 建议方案：资源速率分项明细tooltip

**方案1A（推荐）：资源卡片悬停展开明细**

悬停任意资源卡片时，显示收支分项：
```
⚡ 能量  储量: 342
━━━━━━━━━━━━━━━━━
📈 总产出    +5.0/s
  ATP合成酶×2  +5.0
━━━━━━━━━━━━━━━━━
📉 总消耗    -1.8/s
  建筑运转    -0.8
  🔧 维护费   -0.7
  🦠 人口食物  -0.3
━━━━━━━━━━━━━━━━━
⚖️ 竞争惩罚  ×95%
💰 净速率    +3.0/s
```

**侵入性**：低。只需在资源卡片添加 `mouseenter` 事件，显示 tooltip。数据全部已有（`_maintenanceCost`、`popFoodCost`、`_competitionPenalty`），无需新计算。

**工作量**：约60行JS + 15行CSS

## 问题2：传送带模式下缺少状态提醒

### 📋 现状审计

当前传送带手动连接模式（`_beltConnectMode = true`）的行为：
- ✅ 底部提示变为 `🔗 选择起始建筑（点击格子）`
- ✅ 按钮高亮 `.active` 状态
- ✅ 点击空白区域自动退出
- ❌ **没有超时/闲置提醒** — 如果玩家进入连接模式后去做别的事（比如看科技树），会忘记自己还在连接模式
- ❌ **没有模式指示器** — 屏幕上没有持续可见的"当前模式"标记
- ❌ **其他操作被静默拦截** — 在连接模式下点击建造按钮会被拦截，但没告诉玩家为什么不能用

### 🎯 建议方案：传送带模式状态提醒

**方案2A（推荐）：三层提醒机制**

1. **持续模式指示器** — 屏幕顶部/角落显示浮动标签 `🔗 连接模式` + `按 ESC 退出`，带呼吸动画
2. **闲置超时提醒** — 进入连接模式后30秒无操作，弹出温和提示："还在连接模式哦，按ESC退出"
3. **操作冲突提示** — 在连接模式下点击建造/科技/培养皿等按钮时，提示 "请先完成或退出连接模式"

**侵入性**：很低。不影响任何现有逻辑，纯UI层增强。

**工作量**：约40行JS + 20行CSS

## 问题3：新经济系统对成就系统的影响

### 📋 影响矩阵

#### 直接受影响的现有成就

| 成就 | 影响 | 严重度 | 说明 |
|------|------|--------|------|
| 🏙️ 微型都市（12建筑）| ⚠️ 中等 | 🟡 | 12建筑时维护费约1.2⚡/s+管理开销12%，玩家需要更好布局才能盈利 |
| 🌆 生物大都会（20建筑）| ⚠️ 较大 | 🟠 | 20建筑维护费+72%管理开销，难度显著增加 |
| 📈 效率突破（150%效率）| ⚠️ 轻微 | 🟢 | 竞争惩罚可能拖低有效效率，但不直接影响gEff |
| 🚀 效率翻倍（200%效率）| ⚠️ 轻微 | 🟢 | 同上 |
| ⏱️ 闪电进化（5分钟P2）| ✅ 无影响 | 🟢 | P1无维护费，不受影响 |
| ⏱️ 疾风建造（3分钟8建筑）| ⚠️ 中等 | 🟡 | 快速建8个建筑维护费开始生效，但时间短影响小 |
| 🛡️ 永不断电（10分钟功率≥70%）| ⚠️ 较大 | 🟠 | 维护费消耗葡萄糖+竞争惩罚产出→更容易触发功率下降 |
| ⚖️ 均衡发展（4种建筑）| ✅ 无影响 | 🟢 | 只检查建筑类型数量 |
| 🦠 千菌浩荡（500人口）| ⚠️ 中等 | 🟡 | 人口消耗葡萄糖+维护费消耗葡萄糖，双重压力 |
| 🌍 千菌一心（1000人口）| ⚠️ 较大 | 🟠 | 高人口+高建筑=葡萄糖双重消耗 |

#### 新增的经济成就（3个）

| 成就 | 与经济系统关系 | 设计目的 |
|------|--------------|---------|
| 🔧 入不敷出（铜）| 教学性 | 让玩家意识到维护费的存在 |
| 📊 高效帝国（金）| 挑战性 | 12+建筑全资源正收益=布局功力证明 |
| ⚖️ 供需大师（金）| 挑战性 | P3阶段10+建筑零竞争惩罚=供需平衡 |

### 📊 总体影响评估

**影响程度：中等（可控）**

- **正面影响**：让原本"顺其自然就能拿"的成就变得需要策略，增加了成就含金量
- **潜在风险**：如果调太狠，部分成就从"有挑战"变成"太难了"
- **建议**：暂不改现有成就条件，观察一轮游戏测试数据。如果"生物大都会"（20建筑）完成率低于30%，再考虑降低维护费数值或增加过渡成就

### 💡 建议补充

增加2个**经济过渡成就**作为缓冲：
- 🪙 **收支平衡**（银）：首次在8+建筑时所有资源速率为正（比"高效帝国"低门槛）
- ⚡ **维护减免**（银）：通过代谢回路将维护费降低20%以上（引导玩家发现代谢回路的维护费减免功能）

## 问题4：期货/投资系统 —— 深度评估

### 🎯 设计意图分析

你想要的本质是：**给玩家一个"主动冒险换收益"的渠道**，增加经济博弈深度。

这很好——当前游戏的经济是"确定性"的（放建筑→产出固定速率），加入不确定性能让经济系统更有活力。但"期货"这个词暗示的金融系统可能过于复杂。让我从微生物世界观出发，找到一个更贴合主题、玩家更容易理解的变体。

### 💡 方案D：「基因投资」系统（推荐形态）

#### 核心概念
不叫"期货"，叫**「基因赌注」/「变异投资」**——投入DNA进行基因变异实验，可能产出高价值资源，也可能失败亏损。这完美契合微生物主题。

#### 具体设计

**入口**：P3阶段解锁，在侧边栏增加一个「🧬 变异实验室」面板

**操作流程**：
```
1. 玩家选择一个"变异项目"（从3个随机项目中选1个）
2. 投入资源（DNA + 其他资源作为本金）
3. 等待变异周期（30-90秒倒计时）
4. 结果揭晓：
   - 🟢 大成功（15%）：回报 = 本金 × 2.5~3.0
   - 🟡 小成功（35%）：回报 = 本金 × 1.3~1.8
   - 🟠 平局（25%）：回报 = 本金 × 0.9~1.1
   - 🔴 失败（20%）：回报 = 本金 × 0.3~0.5
   - ⛔ 灾难（5%）：回报 = 0 + 负面效果（效率-3%持续60秒）
```

**与经济系统的深度整合**：

| 整合点 | 机制 | 目的 |
|--------|------|------|
| 维护费压力 → 投资动力 | 维护费吃掉利润→玩家需要高回报来源→投资 | 形成"维护费→投资→回报→扩建→更多维护费"循环 |
| 传送带连接 → 成功率加成 | 变异实验室与周围建筑有传送带连接→成功率+5%/条 | 让传送带布局影响投资收益 |
| 邻接加成 → 回报倍率 | 与特定建筑邻接→回报系数+10% | 让空间布局有投资意义 |
| 资源竞争 → 投资风险 | 当投入的资源处于竞争状态→灾难概率翻倍 | 逼迫玩家在"稳产"和"冒险"间抉择 |
| 同步加成 → 减少灾难率 | 有完美同步的建筑网络→灾难率降为2% | 奖励精妙布局 |

#### 变异项目示例

```
🧬 项目A：「快速增殖」
  投入：8 DNA + 20 能量
  周期：45秒
  成功产出：40~80 葡萄糖 + 15~25 蛋白质
  风险说明：⚠️ 灾难时损失20能量
  
🧬 项目B：「基因重组」  
  投入：12 DNA + 5 蛋白质
  周期：60秒
  成功产出：25~50 DNA + 10~20 生物质
  风险说明：⚠️ 灾难时效率-3%持续60秒
  
🧬 项目C：「量子催化」（P4+）
  投入：15 DNA + 3 QS信号 + 30 能量
  周期：90秒
  成功产出：60~120 能量 + 20~40 生物质 + 8~15 QS
  风险说明：⚠️ 灾难时QS信号归零
```

#### 玩家感知设计（防鸡肋）

这是最关键的部分——怎么让玩家**注意到**并且**愿意用**：

**1. 压力驱动的弹出提示**
```
触发条件：玩家某资源速率为负 且 持续30秒
弹出：「💡 提示：变异实验室可以快速获取 [资源名]！成功率约50%」
频率：每5分钟最多提示一次
```

**2. 视觉存在感**
- 变异实验室在格子上是一个独特的建筑（带旋转DNA动画）
- 实验进行中：建筑格子上有进度环 + 倒计时数字
- 结果揭晓时：全屏闪光 + 大字显示结果（类似赌场开奖的爽感）
- 连续成功时触发"运势"视觉效果（建筑发光越来越亮）

**3. 成就挂钩**
```
🎰 初次下注（铜）：完成第一次变异实验
💰 基因富翁（银）：单次实验回报超过投入200%
🎲 赌神（金）：连续3次实验成功
💀 东山再起（银）：经历灾难后下一次大成功
🧪 稳健投资者（金）：累计10次实验且总回报>总投入
```

**4. 与挑战任务联动**
```
新挑战：「🧬 变异狂潮」— 120秒内完成2次成功的变异实验
奖励：能量150 + DNA40 + 变异实验室永久成功率+2%
```

#### 经济平衡分析

| 指标 | 无投资系统 | 有投资系统 | 变化 |
|------|-----------|-----------|------|
| P3每分钟DNA消耗 | ~2（科技+进化）| ~5（+实验投入）| +150% |
| P3 DNA需求压力 | 低 | 中 | DNA不再过剩 |
| 经济波动性 | 低（线性增长）| 中（有涨有跌）| 更有趣 |
| 维护费应对手段 | 只能优化布局 | 布局+投资双路径 | 策略更丰富 |
| 玩家决策频率 | ~每60秒一次 | ~每40秒一次 | 参与感提升 |

#### 风险评估

| 风险 | 概率 | 后果 | 对策 |
|------|------|------|------|
| 玩家完全不用 | 中 | 功能浪费 | 压力驱动提示+成就引导 |
| 玩家无脑All-in破产 | 低 | 挫败感 | 单次投入上限=储量30%，同时最多1个实验 |
| 期望值不平衡（太赚/太亏）| 中 | 经济崩溃或无人用 | 期望值设计为1.05~1.15（微正），长期有利但短期波动 |
| 与维护费叠加导致过大压力 | 低 | 玩家感觉被双重惩罚 | 实验是主动选择，不是强制消耗 |
| 增加认知负担 | 中 | 新手困惑 | P3才解锁+教学引导+简洁UI |

## 🎯 综合推荐

| 优先级 | 问题 | 推荐方案 | 工作量 | 立即需要？ |
|--------|------|---------|--------|-----------|
| 🔴 P0 | Q1 资源消耗不透明 | 方案1A：资源卡片悬停明细 | ~75行 | **是** — 直接影响玩家理解经济系统 |
| 🟠 P1 | Q2 传送带模式无提醒 | 方案2A：三层提醒机制 | ~60行 | **是** — 简单但体验影响大 |
| 🟡 P2 | Q3 成就系统调整 | 补充2个过渡成就 | ~20行 | 观察数据后决定 |
| 🟡 P2 | Q4 期货/投资系统 | 方案D：变异实验室 | ~340行 | **推荐做** — 但建议单独一轮实施 |

**建议实施顺序**：
1. **先做 Q1 + Q2**（消耗透明化 + 传送带提醒）→ 让现有系统更好用
2. **再做 Q3**（过渡成就）→ 缓冲经济系统对成就的影响
3. **最后做 Q4**（变异实验室）→ 作为独立大功能，需要独立游戏测试
