🔗 资源供给链 — 谁给谁供货？
🌱 碳源采集器 → 🔋 ATP合成酶 → 🧬 简易提取器
▲ 阶段1 基础链：葡萄糖 → 能量 → DNA
🔋 ATP合成酶 → 💨 固氮装置 → ⚗️ 蛋白质工厂 → 🏭 DNA提取器
▲ 阶段2：能量驱动固氮 → 氮源制蛋白 → 高效DNA
🧱 生物膜反应器 ← 🟢+🔵+⚡  →  🗼 群体感应塔 ← ⚡
▲ 阶段3-4：多资源合成生物质 / QS自动加速
💡 关键法则：每个建筑的原料必须由上游建筑通过传送带输送。
没有传送带连接 = 该建筑停产（碳源采集器除外，它由核心供能）。
⛓ 传送带是什么？
传送带是连接两座相邻建筑的资源管道。
上游建筑的产出通过传送带输送到下游建筑作为原料。
阶段1建造时会自动连接传送带；
阶段2+需要你手动点击建筑来连接（更灵活但也更需要规划）。

🌱 阶段1 · 采集
  1. 碳源采集器 — 免费葡萄糖，帝国起点
  2. ATP合成酶 — 有了葡萄糖才能产能量
  3. 简易提取器 — 用葡萄糖+能量合成DNA，进化的钥匙
  4. 研究「纯培养技术」→ 进化到 Lv.2
⚗️ 阶段2 · 代谢
  1. 固氮装置 — 蛋白质需要氮源，这是原料来源
  2. 蛋白质工厂 — 氮源→蛋白质，高级建筑的必需品
  3. 手动连传送带 — 阶段2起不再自动连！点击建筑手动连
  4. 研究「基础代谢学」→ 解锁高效DNA提取器
🔗 阶段3-4 · 物流 & 自动化
  1. 生物膜反应器 → 产出生物质（需要多种原料传送带）
  2. 菌丝运输网 — 全局+10%效率，多造叠加！
  3. 群体感应塔 — QS信号自动加速全产线
  4. 升级传送带 — Lv.1只有75%效率，升级到Lv.3+ 才不亏
