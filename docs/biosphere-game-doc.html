1. 游戏概述与核心循环

微生物帝国是一款网页端放置/增量类游戏，玩家扮演微观生命的引导者， 从最初的原始液泡开始，通过建造采集器、转化站等建筑，逐步构建一个复杂的微生物经济体系。

核心循环

采集资源 → 建造/升级建筑 → 产出更多资源 → 解锁新阶段 → 新建筑/新资源 → 循环
