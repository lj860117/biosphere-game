# 🎮 《微生物帝国》关卡设计评估报告

> **评估人**: 关卡设计师 | **版本**: v0.6.1 | **日期**: 2026-03-18  
> **总体评分**: ⭐⭐⭐⭐ (4/5) — 结构完整、层次丰富的放置策略佳作

---

## 一、关卡总览

### 1.1 五阶段渐进架构

| 阶段 | 名称 | 新资源 | 解锁建筑 | 核心供给 |
|------|------|--------|---------|---------|
| P1 🌱 采集 | 基础资源采集 | 葡萄糖/ATP/DNA | 4种 | 2台碳源 |
| P2 ⚗️ 代谢 | 资源转化链 | 氮源/蛋白质 | 7种 | 4台碳源 |
| P3 🔗 物流 | 空间布局优化 | 生物质 | 6种 | 6台碳源 |
| P4 🤖 自动化 | QS信号系统 | QS信号 | 4种 | 8台碳源 |
| P5 🏛️ 奇观 | 终极目标 | — | 1种(奇观) | 10台碳源 |

**✅ 优**: 每阶段引入1~2新资源+对应建筑群，复杂度阶梯上升。命名和图标直觉传达玩法核心。  
**⚠️ 建议**: P2→P3和P3→P4门槛只需"造出特定建筑"，缺少产出速率的软性挑战。建议加入"ATP产出≥8/s"之类的效率门槛，激励玩家优化而非仅仅解锁。

### 1.2 阶段门槛设计

每次跃迁需三维并行达成——**建筑+科技+进化等级**。结构合理，但后期(P3-P5)可适当增加维度（如资源储量、产出速率、人口数量等）来丰富挑战感。

## 二、资源经济系统

### 2.1 七层转化链

```
核心供能 → 葡萄糖 → ATP → [氮源 → 蛋白质 → DNA]
                                    ↓
                    Phase 3: → 生物质 → Phase 4: → QS信号 → Phase 5: 戴森球
```

**核心数值**:
- 碳源采集器: 0消耗 → 1.5葡萄糖/s（核心免费供能，优秀的起始体验）
- ATP合成酶: 1葡萄糖 → 2.5ATP/s（2.5倍转化，慷慨）
- 简易提取器: 2葡萄糖+1.5ATP → 0.4DNA/s（瓶颈资源，合理的等待墙）

**✅ 优**: "主线清晰、旁路可选"的经典工厂游戏思路。旁路建筑（氨基酸合成仪、核糖体、量子提取器、共振培养箱）提供差异化路径。  
**⚠️ 隐患**: ATP是万能瓶颈资源——几乎所有建筑都消耗ATP，后期可能导致"ATP焦虑"成为唯一优化维度。建议Phase 4+增加不依赖ATP的生产路径。

### 2.2 造价递增

`造价 = 基础 × 1.4^(已建同类数)` — 第5个同类建筑造价达基础的3.84倍。

**✅ 优**: 1.4倍系数恰到好处，在"同类扩产"与"多样化建筑"之间制造有效取舍。

## 三、空间设计

### 3.1 20×20网格（400格）

400格对20种建筑充裕但不浪费。邻接加成鼓励密集布局，实际可利用面积取决于策略而非绝对数量。

### 3.2 邻接系统（38条规则）— ★ 核心空间策略引擎

| 类别 | 规则数 | 代表规则 | 设计意图 |
|------|-------|---------|---------|
| ① 同类协同 | 7条 | 碳源×碳源+10%(叠3) | 鼓励集群布局 |
| ② 上下游供给链 | 23条 | QS直连+18%(最高) | 鼓励供给链紧邻 |
| ③ 增益通用邻接 | 3条 | 菌丝渗透+6%(叠3) | 增益建筑居中辐射 |
| ④ 跨阶段协同 | 4条 | QS塔×生物膜+10% | 混合布局价值 |
| ⑤ 旁路专属 | 9条 | 共振×QS+18% | 旁路融入主线 |

**✅ 亮点**:
- 38条规则创造了海量布局优化空间，追求极致效率可花大量时间规划
- `maxStack`限制防止无脑堆叠
- 增益建筑作为"中央枢纽"增加策略层级
- 上下游邻接与传送带系统形成**双重激励**

**⚠️ 问题**:
1. **信息过载**: 38条对新手过于复杂。建议分阶段解锁邻接规则显示
2. **确定性最优解**: 一旦找到最优组合，空间策略退化为"模板复制"。建议引入随机地形/障碍增加变数

### 3.3 传送带系统

- Phase 1: 自动连接（降低门槛）→ Phase 2+: 手动连接（增加策略）
- 5级升级(+10%/+25%/+45%/+70%/+100%)
- L形路径，不穿过建筑

**✅ 优**: Phase 1自动→Phase 2手动的过渡非常聪明，让新手先体验产线概念再操心布线。传送带可视化（emoji流动动画）增强了"活着的工厂"感觉。

## 四、挑战与平衡性

### 4.1 限时挑战（10个）

| 阶段 | 挑战 | 时限 | 条件 |
|------|------|------|------|
| P1 | 速建达人 | 60s | 建造3个建筑 |
| P1 | 糖分囤积 | 90s | 葡萄糖≥200 |
| P1 | 能量爆发 | 120s | ATP≥300 |
| P2 | 蛋白质冲刺 | 90s | 蛋白质≥50 |
| P2 | 扩建狂潮 | 120s | 8+建筑 |
| P2 | 基因丰收 | 100s | DNA≥80 |
| P3 | 生物质储备 | 120s | 生物质≥30 |
| P3 | 效率狂人 | 150s | 效率≥180% |
| P4 | 信号风暴 | 90s | QS≥20 |
| P4 | 都市扩张 | 180s | 15+建筑 |

**✅ 优**: 挑战覆盖P1-P4，与阶段进度自然匹配。时限设置为60~180秒，符合放置游戏的节奏感。  
**⚠️ 问题**:
1. **缺少Phase 5挑战**: 奇观阶段没有限时挑战，进入终局后缺少紧张感
2. **挑战难度曲线平坦**: P1的"60秒建3个建筑"和P4的"180秒建15个建筑"在对应阶段都属于"正常游玩即可完成"的难度。建议增设"精英挑战"层级（如"60秒内ATP达500，不建新建筑"）

### 4.2 科技互斥分支（5对）

| 二选一 | A选项 | B选项 | 策略深度 |
|--------|-------|-------|---------|
| P1 | 碳源强化(采集+30%) | 能量工程(ATP+25%但耗+15%) | ★★★ 基础资源取舍 |
| P2 | 氮循环(固氮+40%) | 蛋白质工程(蛋白+35%) | ★★★★ 产线侧重选择 |
| P3 | 自适应物流(运输+5%) | 共生网络(人口+200) | ★★★★★ 核心战略分叉 |
| P4 | 信号增幅(QS衰减-60%) | 进化催化(进化效率×2) | ★★★★ 维护vs进攻 |

**✅ 优**: P3的"物流vs人口"是全游戏最有深度的战略分叉——选物流优化空间效率，选人口走人海加成流。这对转生后的重玩价值贡献巨大。  
**⚠️ 建议**: P1的互斥选择(碳源vs能量)影响偏弱，因为两者都在后期被规模化生产淹没。建议让P1互斥更有"长尾效应"（如影响某种后期旁路建筑的解锁条件）。

### 4.3 随机事件系统

- **正面事件**: 11种（资源赠送、效率提升、人口增加）
- **负面事件**: 4种（资源损失、效率下降、人口减少）
- **AB抉择事件**: 6种（如"神秘孢子: 培养获10DNA+8ATP vs 分解获30葡萄糖"）

**✅ 优**: 抉择事件设计精良——每个选项都是"短期资源vs长期效率"的取舍。Phase跟随解锁保证不会出现超出当前理解范围的事件。  
**⚠️ 建议**: 正负比例(11:4)过于友好。放置游戏中适度的"危机感"能提升参与度，建议增加2-3个需要主动应对的负面事件（如"噬菌体入侵！30秒内不建造防御建筑则损失XX"）。

## 五、节奏与心流分析

### 5.1 玩家旅程时间线

```
[0-2min] P1开局 — 建造首个碳源+ATP+DNA提取器
  ↓ "第一道等待墙": 攒DNA做研究(~8s) → 获得即时引导
[2-5min] P1中期 — 研究纯培养, 进化Lv.2, 扩建基础设施
  ↓ 阶段升级高潮！动画反馈+核心外观变化
[5-10min] P2代谢 — 新资源链, 旁路选择, 科技互斥
  ↓ 复杂度跃升, 信息密度高峰
[10-20min] P3物流 — 手动传送带, 邻接优化, 生物质链
  ↓ 空间策略核心期, 最有深度的布局规划
[20-35min] P4自动化 — QS系统, 纳米线, 人口分配
  ↓ 系统全面展开, 多系统联动
[35-60min] P5奇观 — 终极建造, 300秒等待, 大结局
  ↓ 等待期间: 挑战/实验/优化/排行榜
```

### 5.2 节奏评估

**✅ 亮点**:
- P1的"零门槛启动"体验极好——核心免费供能碳源采集器，玩家立刻看到资源增长
- 引导系统（GUIDE）在每个阶段提供4-5条状态感知提示，始终指引"下一步做什么"
- 进化升级+核心外观变化提供强烈的"阶段感"和视觉奖励

**⚠️ 问题**:

1. **P2信息密度骤增**: 从3种资源一下跳到5种，同时解锁7种建筑+科技互斥+传送带手动化+菌群分工。建议将P2拆分为P2a(代谢基础:固氮+蛋白)和P2b(代谢进阶:旁路+互斥)两个子阶段

2. **P5等待期空洞**: 微型戴森球建造需300秒(5分钟)，期间缺乏主动操作内容。建议增加"奇观建造事件"——每60秒触发一次小事件，玩家需要选择（如"检测到结构不稳定！投入20生物质修复/忽略继续"），失败则延长10%工期

3. **功率系统存在感低**: 功率(power)机制设计了5级惩罚(100%/75%/50%/25%/0%)，但缺乏"功率来源管理"的主动交互。玩家几乎无需关注功率——它更像一个后台惩罚而非前台策略

## 六、重玩性评估

### 6.1 转生系统

- **进化因子计算**: 成就数×2 + 阶段×3 + 进化等级×2 + 建筑/5（通关×3倍）
- **13项固定加成**(5~120进化因子): 从"初始资金+"到"终极基因"
- **5种无限循环加成**: 产能/效率/费用/人口/物流（永续追求，成本×1.3~1.4递增）
- **8级被动奖励**: 按转生次数自动叠加

**✅ 优**: 三层转生架构(固定+无限+被动)确保"每次转生都有意义"。无限循环加成特别好——它给了已买完13项固定加成的资深玩家永续追求目标。  
**✅ 优**: 被动奖励的设计节奏好——第1次+5%、第3次+10%产出、第5次-10%费用...每隔几次就有新buff，"再来一轮"的拉力持续存在。

### 6.2 成就系统

- **37个成就**: 4稀有度(铜/银/金/钻石)，12个分类
- **6个里程碑永久buff**: 每5个成就解锁一个（5→效率+5%, 10→费用-8%, ... 30→效率+12%+费用-10%）
- 速度成就（5分钟内到P2、3分钟建8个建筑）→ 鼓励优化开局

**✅ 优**: 里程碑系统很聪明——它把"收集成就"从装饰性行为变成了有实际产出加成的核心循环。玩家会主动规划"这一轮专门冲哪些成就"。

### 6.3 培养皿实验（16个配方）

区域选择+建筑配方→限时buff。冷却90秒。

**✅ 优**: 这是个巧妙的"空间谜题"子系统。需要3×3区域内满足特定建筑组合，把邻接布局从"被动优化"提升为"主动配方目标"。

## 八、Top 10 改进建议（按优先级排序）

### 🔴 高优先级

**1. P2信息密度分流**  
将Phase 2拆为两个子阶段：P2a先解锁基础代谢链(固氮+蛋白)，P2b再开放旁路建筑+科技互斥。减轻新玩家在P2的认知负荷。

**2. P5奇观建造互动化**  
300秒等待期间每60秒触发一次"建造事件"，需要玩家做选择。失败惩罚+成功加速，让终局不只是挂机。

**3. 邻接规则渐进解锁**  
Phase 1只展示7条相关规则(同类协同+P1供给链)，每进阶段追加对应规则。信息面板增加"当前已解锁X/38条邻接规则"提示。

### 🟡 中优先级

**4. 阶段门槛增加效率维度**  
P3→P4要求"ATP净产出≥8/s"，P4→P5要求"QS净产出≥2/s"。逼迫玩家在升阶前真正优化产线。

**5. 增加精英挑战层**  
每个现有挑战增设"精英版"（更苛刻条件+更好奖励），给高手玩家额外追求目标。

**6. 负面事件增强**  
增加2-3个"主动应对型"负面事件——如"噬菌体入侵！30秒内建造防御设施否则损失资源"，让玩家有紧迫操作需求。

**7. ATP单一瓶颈缓解**  
Phase 4引入"生物电池"概念——特定建筑可以直接消耗生物质/QS替代ATP，降低ATP的绝对霸权地位。

### 🟢 低优先级

**8. 随机地形/障碍**  
新开局时随机生成2-4个"不可建造格"（岩石/毒素区域），增加布局多样性，防止"最优模板"固化。

**9. P1科技互斥长尾效应**  
让P1互斥选择影响Phase 3某旁路建筑的解锁——如选"碳源强化"可解锁特殊碳基建筑，选"能量工程"可解锁特殊能量建筑。

**10. 传送带高级玩法**  
增加"交叉传送带"(Phase 4+)允许L形穿越，以及"优先级传送带"允许指定资源优先传输方向，为物流达人提供更高级的优化空间。

## 九、总结

《微生物帝国》作为一款单文件Web放置策略游戏，在系统复杂度和可玩性上达到了令人印象深刻的水准。20种建筑×38条邻接规则×5对互斥科技×传送带布局的排列组合，为一款看似"挂机"的游戏提供了惊人的策略深度。

**最大的设计优势**在于"空间策略"——邻接系统将平面网格从"随便放"变成了"精密规划"，这是同类游戏中少见的深度。

**最需要改进的方向**是"节奏管理"——P2的信息洪水和P5的等待空洞是两个体验凹陷点，优先处理这两个问题将显著提升整体心流体验。

转生系统的三层架构（固定+无限+被动）确保了长期留存——这是一个"玩完一轮想重开"的游戏，这在放置类中是最高评价。

> *"给微生物一个培养皿，它们就能建立帝国——前提是关卡设计师给了它们足够好的成长曲线。"* 🧫
