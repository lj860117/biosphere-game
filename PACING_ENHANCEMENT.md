# 🕹️ 微生物帝国 — 节奏与内容深度增强方案

> **版本**: v1.0 | **日期**: 2026-03-21 | **状态**: 设计提案
>
> 针对玩家反馈"内容太少、游戏时间太短、很快到达终局"的系统性解决方案

---

## §0 问题诊断

### 现状数据摘要

| 维度 | 现有数量 | 说明 |
|---|---|---|
| 阶段 | 5 (P1→P5) | 采集→代谢→物流→自动化→奇观 |
| 建筑 | 21种 | P1:4 / P2:5 / P3:7 / P4:4 / P5:1 |
| 科技 | 13项 | 含4对互斥分支 |
| 突变 | 16种 | 5稀有度,P3+Lv4解锁 |
| 成就 | 37个 | 10分类,6个里程碑奖励 |
| 培养皿配方 | 15个 | P2:6 / P3:4 / P4:2 / P5:1 / 特殊:2 |
| 挑战任务 | 10个 | P1:3 / P2:3 / P3:2 / P4:2 |
| 转生加成 | 13固定 + 6无限 + 8被动 | 总计555🌀固定成本 |

### 核心结论：**不是内容少，是节奏太快**

内容总量其实不少（37成就 + 16突变 + 15配方 + 13科技 + 21建筑），但玩家感觉"内容少"是因为：

1. **进化零阻力** — 进化只需攒资源点一下，没有等待/积累过程
2. **阶段推进无摩擦** — 满足2-3个条件就能直升，不需要在当前阶段横向探索
3. **科技研究太快** — 15~60秒研究时间，几乎感受不到"研究"的过程
4. **P3内容密集但被跳过** — P3有7种建筑+7个配方+16突变，但玩家一心想推P4
5. **没有"停留"动机** — 在每个阶段里没有需要花时间的横向目标
6. **一次通关=全部体验** — 没有足够的重玩差异化

### 首通时间估算（当前）

| 阶段 | 预计时间 | 主要等待 |
|---|---|---|
| P1→P2 | ~8-10分钟 | 攒3🧬 + 建造3种建筑 + 进化到Lv2 |
| P2→P3 | ~10-15分钟 | 研究basicMetab(25s) + 建造蛋白质工厂/DNA提取器 + 进化到Lv3 |
| P3→P4 | ~15-20分钟 | 研究biofilmTech(35s) + 建造生物膜+运输网 + 进化到Lv4 |
| P4→P5 | ~10-15分钟 | 研究quorumSensing(40s) + 建造QS塔 + 进化到Lv5 |
| P5终局 | ~5-8分钟 | 研究dysonTheory(60s) + 奇观建造(300s) |
| **总计** | **~50-70分钟** | |

**目标：将首通时间延长到 2-3小时，同时增加重玩深度**

## §1 进化节奏改革（核心杠杆）

### 问题

当前进化 = `攒够资源 → 点击按钮 → 瞬间升级`。没有任何时间成本或准备过程。

### 方案：进化需要"培养时间"

```
现有:  有资源 → 点击 → 立即升级
改为:  有资源 → 消耗资源 → 开始培养(N秒) → 培养期间可选加速 → 完成进化
```

#### 设计细节

| 进化等级 | 培养时间(秒) | 资源成本(不变) |
|---|---|---|
| Lv1→Lv2 | 30 | 22🧬 |
| Lv2→Lv3 | 60 | 35🧬 + 15🧪 |
| Lv3→Lv4 | 90 | 52🧬 + 22🧪 |
| Lv4→Lv5 | 120 | 76🧬 + 37🧪 + 5🧱 |
| Lv5→Lv6 | 150 | 110🧬 + 56🧪 + 13🧱 |

#### 培养期间互动

- 培养期间建筑继续运转（不是空等）
- 培养进度条显示在进化面板
- **加速机制**：分配"研究个体"到进化培养可加速（与现有种群分配系统整合）
- **随机事件**：培养过程中10%概率触发"进化选择"事件（二选一决策，类似奇观建造事件）

#### 预期效果

- 每次进化增加30-150秒的"培养等待"
- 总进化时间增加约 **7-10分钟**
- 培养期间玩家有事做（优化经济、探索邻接、做培养皿实验）

## §2 阶段推进"软墙"系统

### 问题

当前阶段推进条件太简单（建造1-2种建筑 + 进化等级）。玩家没有动力在每个阶段停留和横向探索。

### 方案：每阶段增加"探索度"条件

在现有条件之外，增加一个"阶段熟练度"指标。达成方式很灵活，但需要一定的横向体验。

#### P1→P2 推进条件（新增）

| 条件 | 现有 | 新增 |
|---|---|---|
| 进化 ≥ Lv2 | ✅ | — |
| 建造碳源采集器 | ✅ | — |
| 建造ATP合成酶 | ✅ | — |
| 建造简易提取器 | ✅ | — |
| 研究纯培养技术 | ✅ | — |
| **P1探索度 ≥ 3** | — | 🆕 |

**P1探索度获取方式（任选其三）：**
- 拥有2+台碳源采集器 (+1)
- 拥有能量缓冲池 (+1)
- 发现3条邻接规则 (+1)
- 完成1个P1挑战 (+1)
- 累计建造5个建筑 (+1)
- 进化到Lv3(超额进化) (+1)

#### P2→P3 推进条件（新增）

| 新增条件 | 说明 |
|---|---|
| **P2探索度 ≥ 4** | 任选以下方式 |

**P2探索度获取方式（任选其四）：**
- 研究一个P2互斥分支科技 (+1)
- 完成2个培养皿实验 (+1)
- 拥有P2b建筑（DNA提取器/氨基酸合成仪/核糖体集群任一）(+1)
- 发现8条邻接规则 (+1)
- 完成1个P2挑战 (+1)
- 拥有10+传送带连接 (+1)
- 累计建造10个建筑 (+1)

#### P3→P4 推进条件（新增）

| 新增条件 | 说明 |
|---|---|
| **P3探索度 ≥ 5** | |

**P3探索度获取方式（任选其五）：**
- 解锁变异实验室 (+1)
- 激活1个突变 (+1)
- 研究一个P3互斥分支科技 (+1)
- 完成3个培养皿实验 (+1)
- 解锁P3高阶配方 (+1)
- 拥有发酵液泡 (+1)
- 发现15条邻接规则 (+1)
- 完成1个P3挑战 (+1)
- 拥有3种不同P3建筑 (+1)

#### P4→P5 推进条件（新增）

| 新增条件 | 说明 |
|---|---|
| **P4探索度 ≥ 4** | |

**P4探索度获取方式（任选其四）：**
- 拥有纳米组装线 (+1)
- 拥有信息素广播站 (+1)
- 拥有共振培养箱 (+1)
- 研究一个P4互斥分支科技 (+1)
- QS信号库存 ≥ 30 (+1)
- 发现20条邻接规则 (+1)
- 完成所有已解锁的P4挑战 (+1)
- 激活3个突变 (+1)

#### 预期效果

- 每个阶段增加 **5-15分钟** 的横向探索时间
- 玩家自然接触到培养皿、挑战、邻接、突变等系统
- 灵活选择方式，不会感觉被强制

## §3 科技研究时间调整

### 问题

当前研究时间15-60秒，感觉像是"点一下等一下"。

### 方案：提升基础研究时间 + 分支科技增加"试验成本"

| 科技 | 原时间 | 新时间 | 变化 |
|---|---|---|---|
| 纯培养技术 | 15s | 25s | +67% |
| 高效采集 | 20s | 40s | +100% |
| 快速代谢 | 20s | 40s | +100% |
| 基础代谢学 | 25s | 45s | +80% |
| 氮循环工程 | 30s | 60s | +100% |
| 蛋白质工程 | 30s | 60s | +100% |
| 生物膜技术 | 35s | 70s | +100% |
| 自适应物流 | 30s | 65s | +117% |
| 共生网络 | 35s | 70s | +100% |
| 群体感应 | 40s | 90s | +125% |
| 信号增幅器 | 35s | 75s | +114% |
| 进化催化剂 | 40s | 80s | +100% |
| 微型戴森理论 | 60s | 120s | +100% |

#### 研究加速机制

保留现有的"研究个体加速"机制（每个研究个体+0.2%）。转生"科研捷径"(-30%)和"超越轮回"(-20%)在新时间基础上生效，确保多周目玩家仍然感觉到进步。

#### 预期效果

- 总科技研究时间从 ~6.5分钟 → ~13.5分钟（+7分钟）
- 转生玩家经过加成后约 ~8分钟（仍快于首通但不至于秒研）

## §4 P3"黄金时代"扩展

### 问题

P3拥有最多的内容（7建筑、4配方、变异实验室）但玩家往往匆匆通过。

### 方案A：P3内部子阶段

将P3分为 P3a（物流基础）和 P3b（物流进阶），类似P2的a/b分区：

| 子阶段 | 需要 | 解锁内容 |
|---|---|---|
| P3a | 研究biofilmTech | 生物膜反应器、菌丝运输网、孢子播种器 |
| P3b | P3探索度 ≥ 3 | 代谢回路、生物质转化炉、噬菌体裂解器、发酵液泡 |

P3b解锁条件示例（任选3）：
- 拥有2种P3a建筑
- 完成1个P3培养皿实验
- 激活1个突变
- 拥有15+传送带
- 发现10条邻接规则

### 方案B：新增2种P3建筑（增量内容）

| 建筑 | 转化比 | 设计目的 |
|---|---|---|
| **质粒扩增器** | 1🧬 + 0.5⚡ → 0.3🧬 + 0.2🧪 + buff(邻接+5%) | DNA→多资源转化 + 邻接增强，鼓励精心布局 |
| **生物传感器** | 0.3📡* + 0.5⚡ → 全局效率+3% | *P3提前少量使用QS概念，为P4做铺垫 |

> 注：生物传感器的QS消耗在P3阶段通过特殊机制获取（如培养皿实验"群体共振"配方产出少量QS），或改为消耗蛋白质。

### 方案C：新增P3专属挑战系统

| 挑战 | 条件 | 时限 | 奖励 |
|---|---|---|---|
| 🧫 膜壁工匠 | 拥有4台生物膜反应器 | 180s | 30🧱 + 效率+5% |
| 🕸️ 菌丝之王 | 菌丝运输网全局加成≥25% | 150s | 20🧱 + 全局产出+8% (120s) |
| 🔗 物流达人 | 20条传送带连接 | 120s | 50⚡ + 20🧬 |
| 🧬 突变猎人 | 培育并激活2个突变 | 180s | 15🧱 + 10🧪 |

#### 预期效果

- P3停留时间从 ~15分钟 → ~25-30分钟
- 玩家自然体验到P3全部系统

## §5 转生差异化与"新游戏+"循环

### 问题

当前转生主要是数值加成，缺乏玩法差异。

### 方案A：转生"变体规则"系统

每次转生后，玩家可以选择1个"变体规则"（可选，不选则普通模式）：

| 变体 | 效果 | 奖励倍率 |
|---|---|---|
| 🔥 **纯粹主义** | 不能使用传送带 | 进化因子 ×2.0 |
| ⚡ **能量危机** | ATP产出-30% | 进化因子 ×1.5 |
| 🧬 **突变狂潮** | 突变槽+3，但突变每60秒随机替换1个 | 进化因子 ×1.3 |
| 🏃 **速通模式** | 20分钟时限 | 进化因子 ×3.0 |
| 🔬 **极简主义** | 最多8个建筑 | 进化因子 ×2.5 |
| 🎲 **混沌培养皿** | 建筑位置随机（不能选择放置格子）| 进化因子 ×1.8 |

### 方案B：新增转生里程碑

| 转生次数 | 永久解锁 |
|---|---|
| 3次 | 🗺️ "布局蓝图"系统 — 可保存/加载建筑布局模板 |
| 5次 | 🧪 "催化剂"消耗品 — 每次转生获得1个随机催化剂（临时强力加成） |
| 10次 | 🌐 "多菌株"模式 — 可同时发展2条科技路线（取消互斥限制） |
| 15次 | ⭐ "创造模式" — 解锁自由探索模式（无限资源，用于测试布局） |

### 方案C：转生成就链

| 成就 | 条件 | 奖励 |
|---|---|---|
| 🔄 十分钟挑战 | 10分钟内通关 | 50🌀 |
| 🔄 无突变通关 | 不使用突变完成游戏 | 40🌀 |
| 🔄 满成就通关 | 单次通关解锁30+成就 | 60🌀 |
| 🔄 极简通关 | 使用≤10个建筑通关 | 80🌀 |
| 🔄 全科技通关 | 研究所有非互斥科技通关 | 35🌀 |

#### 预期效果

- 每种变体 = 1次全新的游戏体验（~2小时）
- 转生成就链提供 5-10次有目标的重玩动力
- 总游戏时长从 ~3-5小时 → ~15-25小时

## §6 即时性"内容填充"方案

以下方案不需要大规模改动，可以快速增加游戏深度感：

### 6.1 随机事件频率提升

| 改动 | 旧值 | 新值 |
|---|---|---|
| 随机事件触发间隔 | 较稀疏 | 每30-45秒触发一次 |
| 选择事件出现率 | 低 | P2起每60秒30%概率触发选择事件 |

### 6.2 邻接发现奖励

当前发现邻接规则只更新图鉴。改为：

- **每发现1条新邻接**：即时获得 5⚡ + 3🟢
- **连续发现（30秒内发现2+条）**：额外 +50% 奖励
- **邻接图鉴里程碑**：每5条发现解锁一个临时buff（+5%对应资源产出，60秒）

### 6.3 建筑升级深化

当前建筑最高等级未限定（或上限较高）。增加：

- **Lv3特化**：建筑升级到Lv3时，弹出二选一"特化路线"
  - 例：ATP合成酶 Lv3 → "超频模式"(产出+40%/消耗+20%) 或 "节能模式"(消耗-30%/产出+10%)
  - 每种建筑有独特的特化选项，增加决策深度

### 6.4 日志/编年史增强

- P3起每30秒自动记录一条"帝国状态快照"
- 通关后展示完整时间线，包括关键决策、资源曲线
- 通关评分增加"探索度"维度（邻接发现%、挑战完成%、培养皿使用次数）

## §9 风险评估

| 风险 | 影响 | 缓解措施 |
|---|---|---|
| 培养等待无聊 | 玩家空等不爽 | 培养期间继续生产+随机事件+种群分配加速 |
| 探索度太强制 | 自由度下降 | 每个阶段提供6-8种获取方式，只需完成3-5种 |
| 科技太慢 | 卡进度烦躁 | 保留研究个体加速+转生科研减免 |
| 老玩家不适应 | 习惯了快速推进 | 转生加成在新节奏下仍然显著加速 |
| P3太长 | 物流阶段疲劳 | P3b是可选深度，不影响P4推进速度 |

## §10 结论

> **一句话总结**：游戏内容本身够用，问题在于玩家"吃太快"。通过增加进化培养时间(§1)、阶段探索度(§2)和科技研究时间(§3)三个核心节奏杠杆，加上P3深度扩展(§4)和转生差异化(§5)，可以在不大量增加新内容的前提下，将游戏体验时长从1小时扩展到2小时（首通）/ 15-20小时（全内容）。

### 推荐实施路径

```
第一步(§3): 调科技研究时间 → 15分钟完成, 立竿见影
第二步(§1): 实施进化培养系统 → 核心节奏改变
第三步(§2): 实施阶段探索度 → 横向深度
第四步(§6): 邻接奖励+建筑特化 → 即时满足感
第五步(§4+§5): P3扩展+转生变体 → 长线深度
```
